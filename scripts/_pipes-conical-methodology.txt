
Расчёт на прочность трубы по ГОСТ 34233.2-2017
Прибавки к расчетной толщине
Прибавка для компенсации коррозии и эрозии
c₁ = 0,5 мм
Прибавка для компенсации минусового допуска
c₂ = 0,3 мм
Сумма технологических прибавок
c₃ = 0 мм
Сумма прибавок к расчётным толщинам стенок
c = c₁ + c₂ + c₃ = 0,8 мм
Исходные данные
Внутренний диаметр трубы
D = 200 мм
Расчётное внутреннее избыточное давление
p = 1,6 МПа
Расчётная температура
T = 200 °C
Допускаемое напряжение при расчётной температуре
[σ] = 140 МПа
Коэффициент прочности продольного сварного шва
φ = 1,0
Результаты расчёта
Расчётная толщина стенки трубы (формула 5.3.1.1)
sₚ = p · D2[σ]·φ - p = 1,6·200 / (2·140·1,0 - 1,6) = 1,14 мм
Исполнительная толщина стенки трубы
s = sₚ + c = 1,14 + 0,8 = 1,94 мм  →  принято s = 2,0 мм
Допускаемое внутреннее давление (формула 5.3.1.2)
[p] = 2[σ]·φ·(s - c)D + (s - c) = 2·140·1,0·1,2 / 201,2 = 1,67 МПа
Условия применимости расчётных формул
s - cD = 1,2 / 200 = 0,006 ≤ 0,1  ✓
p ≤ [p]  →  1,6 МПа ≤ 1,67 МПа  ✓

Расчёт на прочность конической обечайки по ГОСТ 34233.2-2017
Прибавки к расчетной толщине
Прибавка для компенсации коррозии и эрозии
c₁ = 0,5 мм
Прибавка для компенсации минусового допуска
c₂ = 0,3 мм
Сумма технологических прибавок
c₃ = 0 мм
Сумма прибавок к расчётным толщинам стенок
c = c₁ + c₂ + c₃ = 0,8 мм
Исходные данные
Внутренний диаметр большего основания конуса
D = 600 мм
Внутренний диаметр меньшего основания конуса
D₂ = 300 мм
Полуугол при вершине конуса
α = 30°
Расчётное внутреннее избыточное давление
p = 1,6 МПа
Расчётная температура
T = 200 °C
Допускаемое напряжение при расчётной температуре
[σ] = 140 МПа
Коэффициент прочности сварного шва
φ = 1,0
Результаты расчёта
Расчётная толщина стенки конической обечайки (формула 5.4.1.1)
sₚ = p · D2[σ]·φ·cosα - p = 1,6·600 / (2·140·0,866 - 1,6) = 3,95 мм
Исполнительная толщина стенки конической обечайки
s = sₚ + c = 3,95 + 0,8 = 4,75 мм  →  принято s = 5,0 мм
Допускаемое внутреннее давление (формула 5.4.1.2)
[p] = 2[σ]·φ·(s - c)·cosαD + (s - c) = 2·140·4,2·0,866 / 604,2 = 1,69 МПа
Условия применимости расчётных формул
s - cD = 4,2 / 600 = 0,007 ≤ 0,1  ✓
α ≤ 70°  →  30° ≤ 70°  ✓
p ≤ [p]  →  1,6 МПа ≤ 1,69 МПа  ✓
