# GitHub Pages: публичный репозиторий + закрытый доступ к ПНАЭ

## Идея

| Что | Статус |
|-----|--------|
| Репозиторий на GitHub | **Public** (нужно для бесплатного Pages) |
| Сайт GitHub Pages | Публичный URL |
| Справочник ПНАЭ для пользователей | **Только через intech-atom.ru** (iframe на Tilda) |

При прямом открытии ссылки Pages показывается заглушка.  
При встраивании с чужого сайта — отказ.  
Работает только iframe со страниц **intech-atom.ru** / **\*.tilda.ws**.

Защита включена при сборке: `VITE_EMBED_GUARD=true` в `vite.tilda-hosted.config.ts`.

---

## 1. Опубликовать репозиторий на GitHub

1. Создайте репозиторий, например **pnae-steel-handbook** (Public).
2. Загрузите код:

```powershell
cd C:\Users\user\Desktop\stresscalc
git remote add origin https://github.com/intechatom/pnae-steel-handbook.git
git push -u origin main
```

---

## 2. Включить GitHub Pages

1. **Settings → Pages → Build and deployment → Source: GitHub Actions**
2. После push запустится workflow **Deploy handbook to GitHub Pages**
3. **Actions** — дождитесь зелёной галочки
4. **Settings → Pages** — URL вида:

   `https://intechatom.github.io/pnae-steel-handbook/`

Виджеты:

| Путь | Назначение |
|------|------------|
| `/pnae/` | Справочник |
| `/calc1/` | Калькулятор (внутр.) |
| `/calc2/` | Калькулятор (наруж.) |

---

## 3. URL для Tilda

Файл `deploy/github/urls.json` (организация intechatom):

```json
{
  "01-spravochnik": "https://intechatom.github.io/pnae-steel-handbook/pnae/",
  "02-kalkulyator-vnutrennee-davlenie": "https://intechatom.github.io/pnae-steel-handbook/calc1/",
  "03-kalkulyator-naruzhnoe-davlenie": "https://intechatom.github.io/pnae-steel-handbook/calc2/"
}
```

Локально:

```bash
npm run build:tilda:github
```

Код для Tilda: `tilda-hosted/01-spravochnik/TILDA-ВСТАВКА.html`

---

## 4. Вставка в Tilda

Страница **intech-atom.ru/pnae** → блок **T123** → вставить iframe из `TILDA-ВСТАВКА.html` → **Опубликовать**.

---

## 5. Обновления

```bash
git add .
git commit -m "Обновление справочника"
git push
```

CI выполнит `npm run publish:github` и обновит Pages.

---
