Проекты для Tilda (ИНТЕХ-АТОМ)
================================

В папке tilda/ три готовых проекта:

1. 01-spravochnik/          — справочник свойств сталей ПНАЭ
2. 02-kalkulyator-vnutrennee-davlenie/  — калькулятор внутреннего давления
3. 03-kalkulyator-naruzhnoe-davlenie/   — калькулятор наружного давления

В каждой папке:
  index.html                        — одна HTML-страница (CSS и JS внутри)
  data/pnae-steel-properties.json   — база справочника (~1,5 МБ)

Как добавить на Tilda
---------------------

Вариант A — отдельная страница (рекомендуется)
1. Загрузите папку проекта на хостинг Tilda (Файлы / FTP) или свой сервер.
2. Создайте страницу в Tilda.
3. Добавьте блок T123 «HTML» или «Вставка iframe».
4. Укажите URL на index.html из загруженной папки.
   Важно: рядом с index.html должна лежать папка data/ с JSON.

Вариант B — Zero Block (HTML)
1. Откройте index.html в текстовом редакторе.
2. Скопируйте содержимое в Zero Block → HTML.
3. JSON всё равно нужно загрузить отдельно в data/ и путь должен совпадать.

Проверка локально
-----------------
Откройте index.html через локальный сервер (не file://):
  npx serve tilda/01-spravochnik

Пересборка
----------
npm run build:tilda
